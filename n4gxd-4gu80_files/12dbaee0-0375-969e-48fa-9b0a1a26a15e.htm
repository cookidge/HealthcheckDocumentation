Healthcheck Plus 2 Technical Documentation
Window_Main Methods

[This is preliminary documentation and is subject to change.]

The Window_Main type exposes the following members.

Methods
Â  Name Description
BatchDirectory
Returns the batch directory location as a string.
CalculateDedupeFileTotals
Reads through the previously created dedupe stats csv and produces the totals for each dedupe
CalculateDownloadTotals
Reads through the previously created download stats csv and produces the totals for each column.
CalculateFinalDedupeTotals
Reads through the previously created dedupe totals csv and produces the totals for each file
CalculateIndustryTotals
Reads through the previously output industry_suppressions.csv and calculates the sum of each column containing industry suppression list hits.
CheckDedupesExist
Checks the _Merged.xml file for existence of dedupe modules and returns true or false.
CheckDownloadsExist
Reads the _Merged.xml to determine if any downloaded files exist on the Cygnus workflow
CheckInstallations
Checks the installed programs on the computer for GMC Inspire Designer.
CheckISExist
Determines existence of the Cygnus exported industry suppressions file
CompressHealthcheckReport
Deletes all previous zip files in the output directory, before proceeding to ask the user for a valid password. Some basic validation is performed on the password, if it qualifies, the user is no longer asked, and the method then creates a zip file applying the password, and compressing the healthcheck report pdf and the Matrices folder containing each dedupe drops matrix.
ConfirmAnimations
Performs animations after the user clicks confirm on the window.
CreateHealthcheckReport
Determines the correct batch script that needs to be called depending on the files that have been created. Moves the selected batch file to the output directory and runs it. Once the process is complete it checks for the existence of the created report, if it doesn't exist the user is notified and asked to investigate.
CreateReport_Click
The final button that the user uses on the form, it calls the methods that export the basic job information, create and compress the report before sending to the project manager.
CygnusJobDirectory
Returns the Cygnus job directory by amending the merged XML path.
DedupeAnimations
Performs beginning dedupe animations on the window.
EndDedupeAnimations
Performs ending dedupe animatios on the window.
ExportDedupeExamples
Exports examples of the duplicate groups from the supplied dedupe
ExportDedupeInformation
Exports dedupe settings, including rules and criteria for example.
ExportDedupeStatistics
Produces the file statistics for the supplied dedupe, inc total drops, inter, intra etc.
ExportDownloadStatistics
Retrieves the download statistics for the supplied mailing/download file from the accompanying XmlNode, and outputs to file.
ExportDropsMatrix
Exports the matrix of drops for files and suppression files from the supplied dedupe.
ExportFileStructures
Retrieves the file structure for the supplied mailing/download file from the accompanying XmlNode, and outputs to file.
ExportIndustrySuppressions
Reads the industry suppression drops csv exported from the Cygnus workflow, builds a unique list of filenames, and then reads back through the same csv incrementing the total hits for each list, before exporting in a new csv.
ExportSuppressionCosts
Outputs the costs for industry flags and deletes from the totals received as a parameter.
FilesConfirm_Click
Checks that at least one file has been selected, then kicks off the main processing
FilesOnWorkflow
Reads the _Merged.xml file and displays on the form, and downloaded files that exist on the Cygnus workflow
ForenameMatching
Presents the dedupe forename matching level to the user.
FormatName
Removes any characters from a filename that a user may have added during the healthcheck process in Cygnus, to provide more effective matching when creating dedupe totals.
IndustryLevel
Presents the industry suppression forename matching form to the user.
InitializeComponent
InitializeComponent
JobConfirm_Click
Accepts the selected job from the drop down list on the form, and removes unwanted information from the _Merged.xml file to prevent any issues with characters. If industry suppressions exist then it starts the processing methods to create both the count and cost CSVs. The downloaded files that exist on the Cygnus workflow are then displayed to the user on the form.
JobInformation
Exports the basic information of the selected job for use in the report
MailingFileList_MouseDoubleClick
Enables selection of all or none of the downloaded files when the user interacts with the form.
MainFileProcessing
Calls various methods to process files for output. The download information and overseas redirects methods are called. Should dedupes exist then those methods are called here also.
MergedXMLCleaner
Creates a copy of the _Merged.xml file, and removes unecessary information from it, like character, translational or post-suppression dedupe information, that could cause reading problems.
MergedXMLPath
Stores the path to the _Merged.xml file for the Cygnus job.
OutputDirectory
Returns the output folder location as a string.
OverseasRedirects
Retrieves and outputs to file, records that have addresses updated to an overseas address.
ProcessDownloadInformation
Clears the output directory of any pre-existing download files, and proceeds to retrieve the download statistics, from the _Merged.xml file, of each file the user has selected on the form. This is performed by finding the matching process in the _Merged.xml file and passing the process to two methods that export the statistics and file structures respectively.
ProcessIndustrySuppressions
Deletes any previous output files and calls the methods that process both the counts and costs.
ReadInDirectories
RecentJobList_SelectionChanged
Updates the selected job if the job has been selected from the recently reported job list.
RecentlyReportedJobs
Retrieves the ten most recent jobs that have had the healthcheck report created. And adds to the form.
RefreshJobList
Updates the form with a list of available jobs, and adds them to the drop down on the form
RetrieveDedupeInformation
Deletes any pre-existing output CSVs pertaining to dedupes before building a list of dedupes that exist in the Cygnus workflow, and displaying them on the form for the user.
RootOnServer
Returns the root of the selected job output folder on the server.
RunAnimations
Performs initial animations upon running on the window.
RunDedupe_Click
Checks that at least one dedupe module has been selected, loops through each dedupe selected and retrieves the information for it from the _Merged.xml file produced by Cygnus, then calls the methods that export the information, statistics, and drops matrix for each dedupe that was selected. The total methods for each dedupe, and the files overall are then executed.
SelectUser
Converts the current logged in user to a readable first name.
SendReportEmail
Uses the CDO namespace to send an email with the report zip attached, to the PM via the google relay, this relay only works at the Cheltenham Office.
SummaryAnimations
Performs final animations on the window.
